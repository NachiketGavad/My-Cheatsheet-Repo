  1. React with Hook, useState, textform with onClick and onChange event 
    import React,{useState} from 'react'
export default function Textform() {
    const handleUpClick = ()=>{
    console.log("upper clicked");
    setText(text.toUpperCase());
    }

    const handleOnChange = (event)=>{
    console.log("text changed "+ text);
    setText(event.target.value)
    }
    
    const [text, setText] = useState("Enter Text Here");
    return (
    <div className="mb-3">
        <div className="container">
            <label htmlFor="exampleFormControlTextarea1" className="form-label">Example textarea</label>
            <textarea className="form-control" value={text} onChange={handleOnChange} id="exampleFormControlTextarea1" rows="8"></textarea>
            <button className="btn btn-primary" onClick={handleUpClick}>Convert to Uppercase</button>
        </div>  
    </div>
    ) 
}
    1. Import Statements: Import the React library and the useState hook from the react package.
    2. Component Function: Define a functional component named Textform.
    3. useState Hook: Initialize a state variable named text and a function named setText to update the text state. The initial value of text is set to "Enter Text Here".
    4. handleUpClick Function: This function is called when the button is clicked. It logs a message to the console and updates the text state by converting its value to uppercase using the toUpperCase() method.
    5. handleOnChange Function: This function is called when the textarea value changes. It logs a message to the console with the new value of the textarea, and it updates the text state with the new value.
    6. Return Statement: Render a textarea element with a label and a button. The value of the textarea is bound to the text state variable, and its onChange event is handled by the handleOnChange function. The button's onClick event is handled by the handleUpClick function.
    7. In summary, this component provides a textarea where users can input text. They can click a button to convert the text to uppercase. The component utilizes React state to manage the text value and update it dynamically as the user interacts with the textarea and button.
